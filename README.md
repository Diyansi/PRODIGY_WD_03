# Tic-Tac-Toe Game

This is a simple Tic-Tac-Toe game implemented using HTML, CSS, and JavaScript. The game allows you to play in two modes: Player vs Player and Player vs AI.

## Project Structure


- `c.css`: Contains the styles for the game.
- `h.html`: The main HTML file that structures the game.
- `j.js`: The JavaScript file that contains the game logic.
- `README.md`: This file.

## How to Run

1. Clone the repository:
    ```sh
    git clone <repository-url>
    ```
2. Open the `h.html` file in your web browser.

## Game Modes

- **Player vs Player**: Two players can play against each other.
- **Player vs AI**: Play against the computer.

## How to Play

1. Open the game in your browser.
2. Choose a game mode by clicking either "Play with Player" or "Play with computer".
3. Click on the cells to make your move.
4. The game will display a message indicating the winner or if it's a draw.
5. Click the "Reset Game" button to start a new game.

## Features

- Responsive design.
- AI opponent using the Minimax algorithm.
- Simple and intuitive UI.
